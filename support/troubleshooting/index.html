I received one of the following errors "Unable to copy template … to workspace", "Unable to locate template …", “Unable to open … file” or “Unable to run Android build file”, what should I do?

Try re-installing the application as the template files may have become corrupted, if the problem persists please contact us.

I received one of the following errors "Unable to create … folder" or “Unable to copy … into … folder”, what should I do?

Try restarting the application and then recreate your Live Wallpaper, if the problem persists then please contact us.

I received the following error "Java JDK is not installed", what should I do?

Visit the Java SE Downloads page, and download the latest version of the Java SE JDK.

I received the following error "Cannot export Live Wallpaper source code", what should I do?

Please double-check that the destination folder you selected is correct, you can do this by going to the Settings dialog (Menu -> Settings)

I received the following error "please set the destination directory for exporting the source code", what should I do?

You need to set the destination folder where the source code will be exported to, you can do this by going to the Settings dialog (Menu -> Settings)
