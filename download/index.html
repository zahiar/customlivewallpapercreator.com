System Requirements

  • CPU: 800Mhz minimum
  • RAM: 256MB minimum
  • Disk space: 250MB minimum
  • Screen resolution: 1024x768 pixels minimum
  • Operating system: Any Windows® version starting from XP (all 32-bit/64-bit versions supported)

These requirements are not high but if your computer fails to meet them, then CLWC may run slowly or its user experience may be impaired in some other way.

How to Update

Custom Live Wallpaper Creator is continually updated with ground-breaking features and enhancements, therefore we recommend you always uninstall the previous version before installing the new one, to ensure CLWC runs smoothly.

Usually, your setting will remain unchanged when updating to a newer version.
